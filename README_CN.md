## 📦 课程内容

欢迎来到 **Taproot Script Engineering Cohort**！本课程包含以下结构化学习材料：

### 1. **第 1 课**：私钥、公钥和 Taproot 地址编码
- 私钥、公钥和地址编码
- Taproot 的 X-only 公钥
- Bech32m 编码格式
- 完整的 Python 代码示例
- 测试网实践任务
- 提供英文和中文版本

### 2. **第 2 课**：栈操作和 P2PKH 执行
- UTXO 模型深度解析
- 基于栈的执行可视化
- P2PKH 交易构建
- 逐步的脚本执行追踪
- 真实的测试网交易示例
- 提供英文和中文版本

### 3. **第 3 课**：P2SH 多重签名和时间锁
- P2SH 架构和两阶段执行模型
- 构建 2-of-3 多重签名托管合约
- CSV (CheckSequenceVerify) 时间锁
- 栈执行追踪与栈重置机制
- 提供英文和中文版本

### 4. **第 4 课**：SegWit 交易和见证结构
- 交易可延展性和 SegWit 解决方案
- 见证隔离概念
- P2WPKH（原生 SegWit）交易构建
- 交易权重单位和手续费优化
- Taproot 的基础
- 提供英文和中文版本

### 5. **第 5 课**：Taproot 介绍 - Schnorr 签名和密钥调整
- Schnorr 签名的线性特性和密钥聚合
- 密钥调整公式：P' = P + t × G
- X-only 公钥（32 字节）
- 简单的密钥路径 Taproot 交易
- 支付统一性和隐私性
- 提供英文和中文版本

### 6. **第 6 课**：单叶子 Taproot 脚本树
- Taproot 合约的提交-揭示模式
- 带哈希锁的单叶子脚本树
- 控制块（单叶子为 33 字节）
- 双路径花费：密钥路径 vs 脚本路径
- 隐私和效率对比

### 7. **第 7 课**：双叶子 Taproot 脚本树
- 双叶子 Merkle 树结构
- TapLeaf 和 TapBranch 哈希计算
- 带 Merkle 证明的控制块（双叶子为 65 字节）
- Merkle 根计算的字典序排序
- 从同一地址的多个花费路径
- 提供英文和中文版本

### 8. **第 8 课**：四叶子 Taproot 脚本树
- 平衡的四叶子 Merkle 树（两级结构）
- OP_CHECKSIGADD 实现高效多重签名（Tapscript 创新）
- 扩展控制块（97 字节）带多级 Merkle 证明
- 结合 CSV 时间锁和脚本路径
- 五种不同花费路径对比
- 提供英文和中文版本

## 🚀 如何使用这些材料

### 开始学习：
- **8 课内容全部可用**：从基础到高级 Taproot 模式的完整课程材料
- **自主学习**：按自己的节奏学习，每节课预计时长（4-7 小时）
- **先修要求**：每节课都建立在前面的基础上 - 从第 1 课开始按顺序学习
- **双语支持**：大部分课程提供英文和中文版本（在课程列表中标注）

### 课程结构：
- **第 1-2 课**：比特币基础知识（密钥、UTXO、P2PKH）
- **第 3-4 课**：高级脚本（P2SH、SegWit）
- **第 5-8 课**：Taproot 精通（Schnorr、脚本树、企业模式）
- 每节课包含：学习目标、核心概念、代码示例、实践任务和阅读材料

### 学习建议：
- **按顺序完成课程**：每节课都建立在之前的概念上
- **在测试网上实践**：所有代码示例都使用比特币测试网 - 可以安全实验
- **使用两个版本**：阅读英文版本了解技术术语，中文版本深入理解
- **跟随代码操作**：复制并运行 Python 示例，观察概念的实际应用
- **参考书籍**：使用每节课列出的 *Mastering Taproot* 书籍章节进行深入学习
- **参与讨论**：使用 Discord 讨论话题与社区互动
- **寻求帮助**：Office Hours 和 Discord 可用于提问和澄清
