## 📦 课程内容

欢迎来到 **Taproot Script Engineering Cohort**！本课程包含以下结构化学习材料：

### 1. **Course Outline** (`taproot_cohort_outline.md`)
- 完整的12周课程大纲
- 每周课程主题和学习目标
- 先修要求、工具和评估标准
- 帮助你提前了解整个课程安排

### 2. **Lesson 1**: Keys and Addresses (`lesson-01-keys-and-addresses.md`)
- 私钥、公钥和地址编码
- Taproot 的 X-only 公钥
- Bech32m 编码格式
- 完整的 Python 代码示例
- 测试网实践任务
- Discord 讨论话题

### 3. **Lesson 2**: Bitcoin Script Fundamentals (`lesson-02-bitcoin-script-p2pkh.md`)
- UTXO 模型深度解析
- 基于栈的执行可视化
- P2PKH 交易构建
- 逐步的脚本执行追踪
- 真实的测试网交易示例
